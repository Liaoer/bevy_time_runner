# bevy_time_runner
General timing system for the Bevy game engine.
Possible usages:
- Animation
- Rhythm games
- Piano roll in a DAW
- Timeline in a video editor
- Anything elses that use time

This crate is originally a core part of [`bevy_tween`](https://github.com/Multirious/bevy_tween).
